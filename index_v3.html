01

Machine Learning & Deployment

BC Wildfire Danger Prediction System

Machine Learning · Cloud Data Pipeline · AWS

Featured · Live

An end-to-end ML system that predicts daily wildfire danger across British Columbia. It ingests weather and satellite data from multiple sources, runs a trained model over a province-wide grid, and automatically refreshes a live public map every day with no manual intervention. Built with rigorous temporal validation, input-distribution monitoring to catch data drift, and an honest public disclosure of the model's accuracy and limitations.
